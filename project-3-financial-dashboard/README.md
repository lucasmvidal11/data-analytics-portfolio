## Project 3 — Interactive Financial Dashboard

### Objective
Build an interactive financial dashboard to monitor and compare 
the performance of major US banks over the last 12 months.

### Live Dashboard
🔗 [View Live Dashboard](https://[tu-usuario].github.io/
data-analytics-portfolio/project-3-financial-dashboard/
financial_dashboard.html)

### KPIs Tracked
- Current stock price
- 1-year return (%)
- 30-day rolling volatility (%)
- 12-month price range (max/min)

### Companies Analyzed
JPMorgan Chase · Goldman Sachs · Bank of America · 
Morgan Stanley · Citigroup

### Tools Used
Python · Pandas · Plotly · yfinance · GitHub Pages · Google Colab
